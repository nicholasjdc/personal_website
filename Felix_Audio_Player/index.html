This is a simple, sample-based coding language. Within a statement there are three values demarcated by "@" signs: sample name, gain value, and sleep value. Sample name is a string entry, and gain and sleep values are equations whose values are evaluated by conventional javascript syntax. The sleep value is the number of seconds between each repetition of the sample.

Example:

clap@1@1
